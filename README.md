# 個人記帳簿系統 (Personal Accounting System)

## 系統簡介
這是一款專為需要理財規劃的個人用戶所開發的網頁版記帳簿系統。
透過直覺俐落的操作介面，使用者能夠快速進行日常的收入與支出紀錄。系統內建了當前總餘額統計功能，並支援收支的歷史紀錄列表檢視（含刪除功能）與利用圓餅圖清晰呈現的支出分類統計，幫助用戶輕鬆掌握資金流向，養成良好的理財習慣。

## 主要功能
- **總餘額檢視**：首頁直觀顯示當前總餘額、總收入與總支出。
- **隨手記帳**：快速新增單筆收入或支出項目。
- **分類統計**：以 Chart.js 視覺化呈現各分類開銷的佔比（圓餅圖）。
- **歷史清單**：列出過去所有的歷史交易明細，並提供簡易的個別紀錄刪除功能。

## 技術堆疊
本專案採用經典的 MVC 架構開發，確保易於維護與未來擴充性：
- **後端框架**：[Flask](https://flask.palletsprojects.com/) (Python)
- **模板引擎**：[Jinja2](https://jinja.palletsprojects.com/) 加速網頁渲染與資料互動
- **資料庫**：[SQLite](https://www.sqlite.org/index.html) 作為輕量化且便於攜帶的資料儲存方案
- **前端呈現**：HTML5, CSS3, 以及 [Bootstrap 5](https://getbootstrap.com/) 提供響應式 (RWD) 版面

## 快速啟動指南

1. **安裝環境依賴**
   請確保您的系統中已安裝 Python 3。在專案根目錄終端機中執行：
   ```bash
   pip install -r requirements.txt
   ```
2. **初始化資料庫**
   執行輔助指令以建立 SQLite 資料庫 `instance/database.db`：
   ```bash
   python -c "import app; app.init_db()"
   ```
3. **啟動本機伺服器**
   ```bash
   flask run
   ```
   啟動後，使用瀏覽器開啟 `http://127.0.0.1:5000` 即可開始記帳。

## 個人心得

**SDLC (軟體開發生命週期) 與 AI Agent 共同協作的啟發**

在這次專案開發中，除了熟悉 Python 與 Flask 的編程實作，我學習到最寶貴的一課是：**「如何運用 AI Agent 輔助標準的軟體開發生命週期 (SDLC)。」** 

有別於過去總把 AI 單純當作查詢語法的解答機，這次的專案遵循了業界嚴謹的階段性步驟：我們從最模糊的概念出發，先收斂出 **PRD 產品需求文件**確立核心 MVP 目標；接著建立 **系統基礎架構與目錄結構**；然後梳理 **使用者與系統流程圖**、定義出關聯簡單的 **資料庫 Schema**；接著釐清 **API 端點與路由規劃**；等所有設計藍圖都就位之後，才真正著手進行 **前後端全範圍的實作 (Implementation)**。

AI 在每個環節中接手了撰寫基礎樣板與文件生成的繁瑣工作，使我能將專注力放在「主導專案走向」與「檢驗設計邏輯」上。這種「我擔任架構決策者，AI 擔任實作輔助者」的運作模式，不僅大幅縮短了整體開發時間，更幫助我建立起了非常有條理的軟體工程思維。未來若要面對更龐大複雜的系統專案時，這套標準流程與 AI 的結合格外令我感到踏實與自信！
